# Whispr 🔥

A full-stack real-time chat application with Docker & Kubernetes deployment.

## 🚀 Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- DevOps: Docker, Kubernetes (Minikube)

## ⚡ Features
- Secure authentication
- Real-time messaging
- Responsive UI
- Scalable architecture

## 🐳 Docker Setup
docker-compose up --build

## ☸️ Kubernetes Setup
minikube start
kubectl apply -f k8s
minikube service frontend-service
